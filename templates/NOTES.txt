Thank you for installing {{ .Chart.Name }}!

Your release is named {{ .Release.Name }}.

Services deployed:
  - web (frontend, port 80 - main web application)
  - tool-server (API backend, port 8000)
  - langconnect (internal-only, port 8080 - RAG/FastAPI service)
  - postgres (internal-only, port 5432 - pgvector database for langconnect)

To access the web application:

{{- if .Values.web.ingress.enabled }}
{{- range $host := .Values.web.ingress.hosts }}
  {{- range .paths }}
  http{{ if $.Values.web.ingress.tls }}s{{ end }}://{{ $host.host }}{{ .path }}
  {{- end }}
{{- end }}

Make sure the hostname is configured in your DNS or /etc/hosts file.
{{- else }}

  Run this command to forward the web frontend port:
  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "summitlabs.fullname" . }}-web 8080:80

  Then visit http://127.0.0.1:8080
{{- end }}

To access tool-server directly:

{{- if .Values.toolServer.ingress.enabled }}
{{- range $host := .Values.toolServer.ingress.hosts }}
  {{- range .paths }}
  http{{ if $.Values.toolServer.ingress.tls }}s{{ end }}://{{ $host.host }}{{ .path }}
  {{- end }}
{{- end }}
{{- else }}
  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "summitlabs.fullname" . }}-tool-server 8000:8000
{{- end }}

To check the status of your deployment:

  kubectl get pods -n {{ .Release.Namespace }}
  kubectl get svc -n {{ .Release.Namespace }}
{{- if .Values.toolServer.ingress.enabled }}
  kubectl get ingress -n {{ .Release.Namespace }}
{{- end }}

To view logs:

  # web frontend logs
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=web

  # tool-server logs
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=tool-server

  # langconnect logs
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=langconnect

  # postgres logs
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=postgres

To test connectivity between services:

  kubectl exec -n {{ .Release.Namespace }} deployment/{{ include "summitlabs.fullname" . }}-tool-server -- \
    curl http://{{ include "summitlabs.fullname" . }}-langconnect:8080/
